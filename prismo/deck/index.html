Solution

A config file
for brand truth.

brand-verify.json is a machine-readable file where brands declare their identity, voice, claims, and corrections.

Like robots.txt told search engines what to crawl, brand-verify.json tells AI agents how to talk about a brand.

The file is the standard. Prismo is the verification layer that checks AI output against it.

// brand-verify.json
{
  "name": "Acme Corp",
  "voice": {
    "tone": "direct, warm",
    "vocabulary": {
      "preferred": ["craft", "build"],
      "avoid": ["cheap", "basic"]
    }
  },
  "corrections": [{
    "wrong": "Founded in 2019",
    "right": "Founded in 2015",
    "severity": "critical"
  }]
}
How It Works

Three steps.

1

Declare

Brand creates a brand-verify.json file with their identity, claims, corrections, and voice guidelines.

2

Verify

Prismo checks what AI models say against the brand's declared truth. Detects factual errors, tonal bias, and vocabulary violations.

3

Correct

MCP server feeds corrections directly to AI assistants at query time. Brands fix misrepresentation at the source.
